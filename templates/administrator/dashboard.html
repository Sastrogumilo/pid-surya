Hibridisasi Evolution Strategy Dengan Particle Swarm Optimization Sebagai Metode Tuning Parameter Kendali PID Pada Pengendalian Kecepatan Motor DC

Sistem

Sistem kendali kecepatan motor DC menggunakan metode tuning metaheuristic sehingga diperoleh diagram respon keluaran dan parameter kendali PID yang optimal.

Metode

Metode tuning metaheuristic yang digunakan adalah hibridisasi Evolution Strategy dengan Particle Swarm Optimization untuk mencari Parameter Kendali PID pada Pengendalian Kecepatan Motor DC.

Menu


Menu dashboard merupakan halaman awal sistem yang menampilkan judul penelitian dan penjelasan dari sistem.

Menu Inisialisasi Parameter merupakan halaman untuk input parameter Motor DC.

Menu PID merupakan halaman untuk input parameter pada metode PID dan menampilkan hasil diagram respon keluaran.

Menu PID dengan ES merupakan halaman untuk input parameter pada metode PID-Evolution Strategy dan menampilkan hasil diagram respon keluaran dengan nilai Kp, Ki, Kd terbaik.

Menu PID dengan PSO merupakan halaman untuk input parameter pada metode PID-Particle Swarm Optimization dan menampilkan hasil diagram respon keluaran dengan nilai Kp, Ki, Kd terbaik.

Menu PID dengan ES+PSO merupakan halaman untuk input parameter pada metode PID-Evolution Strategy dengan Particle Swarm Optimization dan menampilkan hasil diagram respon keluaran.

Rangkuman merupakan halaman untuk menampilkan rangkuman semua hasil diagram respon keluaran semua metode.

Profil merupakan halaman untuk menampilkan profil penulis.
